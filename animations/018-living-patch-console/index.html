Living Patch Console

A living instrument instead of a static interface mockup: one patch rail changes how the field thinks, shifting red-white dot polarity, wave rounding, and membrane pressure inside the same responsive world.

Press the rail to rewire the behavior. Drag through the screen to bruise and lift the local tissue. The controls should feel like they are touching the organism directly, not operating a separate widget.
