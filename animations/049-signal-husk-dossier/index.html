Signal Husk Dossier

A darker illustrated control folio for one grounded three-dot organism, with embedded seams, bead feelers, and clocked handoff windows so the page reads like a usable specimen dossier instead of another detached dashboard or pale greenhouse atlas.

Phase overrides

The page runs on a repeating relay cycle: hold, sift, then flare. Move across the board to bias the current window locally. Press or drag to make the husk brace harder during the active handoff.

Why this branch

Recent feedback kept asking for something more design-usable and closer to a real HTML interface image, while also rejecting creature anatomy that feels implausible or decorative. This page keeps a stronger dossier/interface language but anchors it in one body with believable weight, mirrored support, and timed phase handoffs instead of masks, extra limbs, or detached panels.

Idea

A dark browser-native dossier where one grounded signal husk sits in a specimen plate, and the interface logic is embedded in its seams, feelers, stability marks, and side ledgers, with scheduled release windows instead of a separate dashboard.

Interaction

A repeating relay cycle hands the body through hold, sift, and flare phases. Pointer position only biases the active window locally, while pressing the board adds a short brace pulse that widens the stance and spikes the bead threads in place.

Next

Push this dossier language into a larger field manual with a few neighboring species, or keep the same single-body page and make the relay timetable editable through text or live external events.
